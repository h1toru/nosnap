# nosnap
A bash script to remove Snap package manager completely from Ubuntu.

### Usage
```bash
curl -fsSL https://raw.githubusercontent.com/h1toru/nosnap/master/nosnap.sh | bash
```

### Compatibility
- [x] Ubuntu 23.04
- [x] Ubuntu 22.10
- [x] Ubuntu 22.04.xx LTS
